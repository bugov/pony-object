import

sub import

This function will runs on each use of this module.  It changes caller - adds new keywords, makes caller more strict and modern, create from simple package almost normal class.  Also it provides some useful methods.

Don’t forget: it’s still OOP with blessed refs, but now it looks better - more sugar for your code.
